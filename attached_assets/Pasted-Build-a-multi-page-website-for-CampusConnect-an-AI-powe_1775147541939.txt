Build a multi-page website for "CampusConnect" — an AI-powered student networking and collaboration platform.

Tech stack: React + Vite + TypeScript + Tailwind CSS v4 + React Router.

Design style: Pure black background across all pages, cinematic, futuristic, premium, minimal. Should feel like a real startup product (Apple x Notion x Gen-Z campus vibe).

---

🌐 ROUTING STRUCTURE

Use React Router and create these pages:

* "/" → Home
* "/features" → Features page
* "/how-it-works" → How It Works
* "/feed" → Campus Feed Preview (IMPORTANT)
* "/about" → About / Vision
* "/join" → Get Started

Navbar should be consistent across all pages.

---

🎨 DESIGN SYSTEM (KEEP EXACT)

Fonts:
* Instrument Serif (italic) → headings
* Barlow → body

Theme:
* Black background everywhere
* White text
* Glassmorphism (liquid glass effect)
* Smooth animations (Framer Motion)
